An \eval for TeX

 Being an uncomplicated way of using Scheme 
 to specify typographically difficult elements 
 such as pictures in one’s TeX document 
 and have them automatically convert to appropriate and 
 pleasing output in both the printed and online 
 versions of the document 

 Download Version 20090506 
 Dorai Sitaram 

I shall first describe how to set up Scheme as an extension language for TeX [3, 4], then show how this mechanism can add powerful features to TeX, using as example a high-level picture language.
